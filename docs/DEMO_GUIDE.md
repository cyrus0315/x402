## 🚀 3 分钟演示脚本

### 第 1 分钟：项目介绍

> "PayPerInsight 是一个去中心化的知识付费平台，基于 x402 协议和 Monad 构建。"

**核心亮点**:
- ⚡ **即时微支付** - 无需订阅，按需付费
- 🎨 **解锁即 NFT** - 付费后自动铸造 NFT 作为凭证
- 📈 **动态定价** - 早期用户享受更低价格
- 💰 **推荐分成** - 分享链接赚取 10% 佣金

## 🗣️ 演讲要点

### 问题陈述
> "传统知识付费平台存在高抽成、订阅锁定、支付摩擦等问题。"

### 解决方案
> "PayPerInsight 利用 x402 协议实现即时微支付，用户无需注册、无需订阅，直接用钱包支付解锁内容。"

### 技术优势
> "基于 Monad 的高性能，我们可以实现每秒数千次的微支付交易，Gas 费用极低。"

### 商业模式
> "平台收取 5% 费用，创作者获得 85%，推荐人获得 10%，形成正向激励循环。"

## ❓ 可能的评委问题

### Q: 为什么选择 x402 而不是传统支付？

> "x402 是 HTTP 原生的支付协议，用户无需注册、无需 API 密钥，只需钱包签名即可完成支付。这大大降低了使用门槛。"

### Q: 动态定价的目的是什么？

> "动态定价创造了 FOMO 效应，激励用户尽早解锁。同时也让创作者的优质内容获得更高回报。"

### Q: 为什么解锁要铸造 NFT？

> "NFT 作为访问凭证，可以转让、出售。这意味着用户不仅获得内容，还获得了一个可交易的资产。"

### Q: Monad 相比其他链的优势？

> "Monad 提供 10,000+ TPS 和亚秒级确认，非常适合高频微支付场景。同时完全 EVM 兼容，开发成本低。"
